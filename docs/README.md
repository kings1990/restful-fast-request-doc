### 用插件代替手工 :100:
```
1.再也不用手写方法参数了
2.再也不用关注如何传参了
3.再也不用在跳出IDEA调试接口了
4.API和代码关联起来了,不用到处找了
```

::: tip 注意  
请确保你的 IDEA版本 版本 >= 2020.3+  

当然请如果插件有更新，请确保更新成最新版本
:::
